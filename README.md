# Fable Sync Repository

This repository contains the plugin manifest and distribution metadata for the Fable Sync FFXIV plugin.

## Structure

- `plogonmaster.json` - Master plugin list for Dalamud plugin installer
- `FableSync/FableSync.json` - Individual plugin manifest
- `LICENSE` - Repository license

## Plugin Information

**Name:** FableSync Sync  
**Author:** Wrepsk  
**Description:** A character synchronization plugin for FFXIV that synchronizes character mods and appearance with other players.

## Installation

Add this repository to your Dalamud plugin installer:

```
https://raw.githubusercontent.com/Wrepsk/repo/main/plogonmaster.json
```
